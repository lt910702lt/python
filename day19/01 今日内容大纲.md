01 昨日内容回顾
    继承：
        为什么要有继承？
           优化代码
           节省代码
           提到代码的复用性
           提高代码的维护性
           让类与类之间发生关系
        组合：让对象与对象发生关系
    子类中实现父类的方法，又实现自己的方法：
        super().父类方法名(自传self)
        父类名.父类方法名(参数)
    单继承、多继承
    class A:
        pass
    class B(A,C,D):
        pass
    新式类：
        继承object的类都叫新式类
        python3中所有的类默认都继承object，所以python3中全部都是新式类
        新式类：广度优先    
    经典类：
        不继承object的类就是经典类
        python2中所有的类都默认不继承object，也可以自定义继承object
        经典类：深度优先
